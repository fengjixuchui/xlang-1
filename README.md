# XLang
百度搜索了一下，能跨平台支持的语言少之又少，不是脚本语言就是还要进行少量修改的。     
所以就写了这个东西出来...     
# 语法
一个最简单的HelloWorld程序就长这样的
```
import x.console;
import x.string;
namespace helloworld{
	 public class hello{
		public static void Main(string args[]){
			console.out("Hello,World");
		}
	}
}
```
